# DeliPedia Frontend

## 개요
- DeliPedia 프론트엔드
- React.js를 TypeScript 기반으로 작업
- 화면 퍼블리싱 완료 후 Restful API 붙이는 작업 진행

## 프로젝트 실행

`yarn start` 실행 시 webpack에 의해 개발자 모드로 빌드됨.
`npm start`로 실행하지 말 것.
